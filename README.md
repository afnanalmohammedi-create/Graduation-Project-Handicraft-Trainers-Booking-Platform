Handicraft Trainers Booking Platform
Graduation Project - Management Information Systems (MIS)
This is a web-based platform designed to connect clients with specialized handicraft instructors. The system allows users to browse different craft categories, view instructor profiles, and manage bookings through an integrated interface.

Technical Stack
Frontend: HTML5, CSS3, JavaScript.

Backend: PHP (Server-side logic).

Database: MySQL (Relational Database Management).

Assets: UI/UX designed with custom icons and images.

Key Features
User Authentication: Secure sign-in and registration system (Signin.php).

Instructor Directory: Specialized pages for different trainers and crafts.

Database Integration: Seamless linking between the web interface and the backend database to manage bookings and user data.

Responsive Design: User-friendly interface across different pages.

Database Contribution
As an MIS student, I focused on the Systems Analysis and Design phase, ensuring that the database schema is optimized for data integrity and smooth user transactions.
